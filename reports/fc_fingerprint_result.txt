As an additional external validation, we computed the Spearman rank correlation between the Fisher-z-transformed FC values of all valid edges and a canonical network-membership reference (1 = same-network pair, 0 = cross-network pair; 3093 within-network and 16807 between-network edges). The observed correlation (Spearman rho = 0.32, 95% CI [0.31, 0.34]) confirms that FC values are systematically structured according to canonical network boundaries: same-network ROI pairs have substantially higher functional connectivity than cross-network pairs across the full edge distribution. Within-network FC was highest for the visual network (Fisher-z M = 0.72) and somatomotor network (M = 0.45), consistent with the well-documented strong local synchrony in these systems.
